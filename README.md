# CORE Electronics — Product Landing Page

A premium, minimalist landing page inspired by modern tech aesthetics. This project features **Glassmorphism UI**, responsive layouts, and smooth navigation.

## 🚀 Demo
https://laddtnov.github.io/glassmorphism-product-page

## ✨ Key Features
* **Glassmorphism Design:** Uses `backdrop-filter` for high-end visual effects.
* **Responsive Layout:** Fully optimized for mobile, tablet, and desktop using CSS Flexbox and Media Queries.
* **Smooth UX:** Integrated smooth scrolling and interactive hover states for all UI elements.
* **Validated:** Meets all 16/16 User Stories for the freeCodeCamp "Product Landing Page" certification.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure.
* **CSS3:** Flexbox, Grid, Glassmorphism, Animations.
* **Typography:** Inter (via Google Fonts).
